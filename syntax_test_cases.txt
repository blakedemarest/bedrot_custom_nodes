BEDROT Syntax Highlighter - Comprehensive Test Cases
================================================================================

This file contains 100+ edge cases for testing the regex-based syntax
highlighting in bedrot_highlighter.js. Each category tests specific
tokenizer behaviors and potential failure modes.

Legend:
  [N]         = Flag token (activates flag N)
  [K: ...]    = Positive conditional (shown when flag K active)
  [-K: ...]   = Negative conditional (shown when flag K NOT active)
  [-N]        = Invalid bare negative (should be red/removed)
  (text:1.2)  = CLIP weight syntax
  {a|b|c}     = Random choice block
